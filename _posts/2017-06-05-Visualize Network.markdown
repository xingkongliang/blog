# Netscope
Netscope是一个基于web的可视化网络结构的工具。它当前支持Caffe的prototxt格式。

Netscope主页：[这里](http://ethereon.github.io/netscope/quickstart.html)

Netscope Github: [这里](https://github.com/ethereon/netscope)
